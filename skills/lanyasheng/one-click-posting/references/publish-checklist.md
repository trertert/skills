# 发布前后检查清单

## 发布前（必须全绿）
- [ ] 标题、正文、标签、平台列表已固定
- [ ] 来源可追溯；待核实信息已标注
- [ ] 去 AI 味检查完成（必要时已重写）
- [ ] 用户明确同意发布（硬门禁）
- [ ] 发布包已生成（JSON）

## 发布中（证据留存）
- [ ] 每个平台保留关键截图
- [ ] 如遇异常，记录错误信息与重试动作

## 发布后（复盘）
- [ ] 记录结果：成功/失败/审核中
- [ ] 记录链接或笔记 ID
- [ ] 回填首小时指标（曝光/赞/评/藏/转）
- [ ] 归档到 `knowledge/daily/YYYY-MM-DD/`
- [ ] 同步到 `memory/YYYY-MM-DD.md`
